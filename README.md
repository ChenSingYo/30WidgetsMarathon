## **Setup**

下載npm相關套件（devDependencies）
```bash
npm install
```

利用vite打包出dist資料夾
```bash
npm run build
```

啟動專案
```
npm run dev
```

點擊localhost:3000，於瀏覽器檢視專案內容
```
http://localhost:3000
```

## **Mainly Build With**

Vite

Vue

tailwindcss 2.2
